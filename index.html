About This Website

I enjoy learning about computers and I have wanted to build a website since I was in college. As a solo hobbyist I have found it too burdensome and time consuming to undertake programming projects of any nontrivial size. I recently wanted to write a small program and was amazed at how quickly I was able to build it with an AI copilot in VSCode. This website is part of my experiments using an AI copilot for pair programming.
